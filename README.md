# briskGaugeAC

App for Assetto Corsa. Displays essential information in Škoda fabia lookalike instrument cluster.

## How to install

Merge the *"assettocorsa"* folder with the *"assettocorsa"* instalation folder (Usually *"Steam\steamapps\common\assettocorsa"*).

## How to use
Enable the app in the *"Options -> General -> UI Modules"* menu. Run the app from the *"Apps"* ingame sidemenu.
